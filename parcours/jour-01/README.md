À la fin de la journée, tu dois être capable de :

## Exercice 1 — Premier programme

Crée un fichier `bonjour_finance.py` contenant :

```python
print("Bonjour, bienvenue dans l'équipe finance !")
```

Lance le programme avec :

```bash
python bonjour_finance.py
```
